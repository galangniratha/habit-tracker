# Habit Tracker Web App

Aplikasi Habit Tracker berbasis web menggunakan PHP dan MySQL.

## Fitur
- Login dan Register
- Tambah kebiasaan
- Checklist habit
- Hapus habit
- Grafik evaluasi kebiasaan

## Teknologi
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Chart.js

## Cara Menjalankan
1. Install XAMPP
2. Copy folder project ke htdocs
3. Import database ke phpMyAdmin
4. Jalankan di browser

http://localhost/HabbitTracker
